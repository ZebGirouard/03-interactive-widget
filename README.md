# 03 Interactive Widget

## Goal
Use JavaScript to connect a button click to visible UI changes so the DOM stops feeling magical.

## Start Here
1. Open `index.html` in your browser.
2. Wire up the missing event listeners in `script.js`.
3. Make the counter update, reset, and toggle themes.

There is no build step.

## Stretch Goals
- Add a decrement button.
- Prevent the count from going below zero.
- Save the theme in `localStorage`.
