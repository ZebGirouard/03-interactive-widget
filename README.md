# 03 Interactive Widget

## Goal
Use JavaScript to connect button clicks to a visible checklist so the DOM stops feeling magical.

## Start Here
1. Open `index.html` in your browser.
2. In `script.js`, look at the one completed click listener example.
3. Use that same pattern to connect the other buttons to the helper functions that match their jobs.

## What You Are Practicing
- A listener waits for something to happen on the page, like a button click.
- A function call runs a named block of code, like `toggleTheme()`.
- The `<script src="script.js"></script>` tag is how this page loads its JavaScript file.

## Stretch Goals
- Add a previous-step button.
- Prevent the current step from going below zero or above the last step.
- Save the theme in `localStorage`.
